# PhonePe UI Clone

This project is a simple frontend clone of the PhonePe mobile app interface.

## Tech Used
- HTML
- CSS
- JavaScript

## Features
- Wallet balance section
- Payment options
- Recent transactions
- Bottom navigation

## Purpose
Created for learning frontend development and UI cloning.
